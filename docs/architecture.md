# Architecture

## Positionnement
`llm-wiki-minimal` est une couche de pré-consolidation documentaire.

Il ne remplace pas la mémoire canonique d’un repo consommateur.

## Flux
Sources ciblées → extraction structurée → fiches wiki minimales → revue humaine → promotion éventuelle vers le canonique.

## Principes
- fiches courtes
- provenance explicite
- liens explicites
- statuts visibles
- promotion humaine seulement
